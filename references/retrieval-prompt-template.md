# 外部AI助手检索提示词模板

## 使用场景

用户需要通过Kimi等带专利知识库的AI助手进行专利检索时，生成可直接粘贴的提示词。

## 提示词结构

### 核心专利检索提示词

```
## 任务

我需要检索以下X篇中国专利的摘要和权利要求书内容，用于评估一项[技术领域]技术的新颖性。请逐篇检索并回答每篇专利中是否涉及我列出的具体技术问题。

## 检索要求

请使用你能访问的专业专利数据库，检索以下专利的摘要和权利要求书内容。

---

### 专利1：[申请人] [专利号]

- **申请人**：[申请人名称]
- **公开日期**：[日期]
- **专利标题（已知）**：[标题简述]

**需要确认的技术问题**：
1. [具体问题1]
2. [具体问题2]
3. [具体问题3]

---

[重复上述格式]

## 输出格式要求

对每篇专利，请按以下格式输出：

```
【专利X】XXX CNXXXXXXXXXA
- 检索状态：[成功 / 未找到 / 仅找到部分信息]
- 摘要核心内容：[100字以内概括]
- 权利要求书关键内容：[概括独立权利要求的技术特征]
- 问题1回答：[是/否/未明确，附依据]
- 问题2回答：[是/否/未明确，附依据]
- 问题3回答：[是/否/未明确，附依据]
- 与本发明的相关性评估：[高/中/低，简要说明]
```

## 补充通用检索

在完成上述专利检索后，请额外检索以下技术方向的专利：

1. **检索方向A**：[方向描述]
   - 检索式：（关键词组合）

[列出最相关的1-2篇专利的专利号、标题和简要摘要]

## 重要提示

- 如果某篇专利在你的数据库中无法获取全文，请明确告知"未找到"，不要猜测
- 如果摘要中的信息不足以回答某个问题，请标注"信息不足"
- 所有结论必须有专利文本中的具体语句作为依据
```

### 补充检索提示词

当需要针对特定技术特征进行补充检索时，使用更聚焦的提示词：

```
## 补充检索任务

请检索涉及以下技术特征的中国专利：
- 技术领域：[具体领域]
- 核心特征：[需要检索的特征1]、[特征2]、[特征3]
- 排除方向：[已确认不相关的方向]

请列出找到的专利号、申请人、标题和简要摘要，按相关性排序。
```
